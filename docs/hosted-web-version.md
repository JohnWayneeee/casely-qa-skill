# Casely Hosted Web Version

Casely started as an open-source QA skill for generating test cases from requirements.

The hosted web version is for QA teams that want the same workflow in a browser:

- Upload requirements and product documents.
- Add examples of existing test cases.
- Generate functional, negative, boundary, smoke, and integration cases.
- Review generated coverage with the team.
- Export review-ready test cases for tools like TestRail, Qase, Excel, or Markdown.

The open-source skill is a good fit when you want to run the workflow locally in your AI IDE.
The hosted web version is a better fit when your team wants uploads, review flow, and no local setup.
